Element-wise array addition

November 12, 2022

Use a for-loop to add elements in arrays.

let x: [Float] = [3, 4.5, 8]
let y: [Float] = [2, 1.2, 7.4]

var z = [Float](repeating: 0, count: x.count)

for i in 0..&lt;x.count {
    z[i] = x[i] + y[i]
}

// z is [5.0, 5.7, 15.4]
