Elapsed time

Use DispatchTime to calculate the elapsed time for some Swift code. In the example below, the sleep function causes a delay of 4 seconds which is calculated by the elapsed variable.


// example.swift

import Cocoa

let tic = DispatchTime.now().uptimeNanoseconds

sleep(4)

let toc = DispatchTime.now().uptimeNanoseconds

let elapsed = (toc - tic) / 1_000_000_000
print("Elapsed time is \(elapsed) seconds")
Running the above example in the terminal gives the following output: 

$ swift example.swift
Elapsed time is 4 seconds
